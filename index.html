PROJECTS

Sktech App

Technologies:
HTML, CSS, JavaScript

Description:

This app utilizes Vanilla Javascript DOM manipulation to allow the user to "pixel paint" on the screen. The paintbrush allows you to paint random colors by swiping across the squares, while the eraser gives more thurrough control by allowing you to click on squares you want reset. The squares are created dynamicly, so you can change the size of the board to choose your pixel art size.
